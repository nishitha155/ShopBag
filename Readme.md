# ShopBag

## Frontend
- Navigate to the frontend directory: `cd frontend`
- Install dependencies: `npm install`
- Run development server: `npm run dev`

## Backend
- Navigate to the backend directory: `cd backend`
- Install dependencies: `npm install`
- Run the server: `npm start`
